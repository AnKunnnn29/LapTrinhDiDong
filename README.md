# lapNhom

Đây là một dự án ứng dụng Android đơn giản bao gồm chức năng đăng nhập và đăng ký.

## Tính năng

*   Màn hình đăng nhập
*   Màn hình đăng ký
*   Điều hướng cơ bản giữa các màn hình

## Cài đặt

1.  Clone repository này.
2.  Mở dự án trong Android Studio.
3.  Chạy ứng dụng trên máy ảo hoặc thiết bị thật.
